Here you can place custom marquees by system folderto customize system marquees.

Create a folder named by system theme like snes, megadrive, mastersystem etc.. or duplicate existing folder and rename it.


In each folder you can place files named:"11, 12, 13, 14, 31, 32, 33, and 34" with .png extension.

In theme this will display like this:		11	12	13	14

						    x	    x	    x

						31	32	33	34
